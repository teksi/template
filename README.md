# template
DEV template for any future TEKSI module

## Module code architecture

There are several folders in the base code repository to dispatch code for the 

* [.github](.github) for workflows and issue templates
* [datamodel](datamodel) for all code related to the datamodel
* [docs](docs) for all assets related to the documentation
* [plugin](plugin) for all code related to the QGIS business plugin
* [project](project) for all code related to the QGIS .qgs project file
